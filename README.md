# 💻 Mini Web Development Projects

This repository contains a collection of mini projects built using HTML, CSS, and JavaScript.  
The goal of these projects is to strengthen core frontend development skills through hands-on practice.

Each project focuses on key concepts such as DOM manipulation, event handling, responsive design, and user interaction.

---

## 🚀 Projects Included

- 🌦️ Weather App – Search weather by city name  
- ✅ Todo App – Add, delete, and manage tasks  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 🎯 Learning Outcomes

- Improved understanding of JavaScript fundamentals  
- Hands-on experience with real-world UI behavior  
- Better grasp of DOM manipulation and event handling  
- Practice building responsive and interactive interfaces
